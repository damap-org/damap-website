# Recommended Repositories

DAMAP allows administrators to configure repository recommendations that are presented to users.
This can be done easily by using the search field and filters.

![Searching and adding repositories](../repositories-media/search-repository.jpg)

After an administrator has added repositories, they will appear as recommended options for all users:

![Recommended repositories](../repositories-media/recommended-repository.jpg)

!!! tip
    Deleting a recommended repository does not remove it from any existing DMPs that use it.
