# Custom Banner

DAMAP allows administrators to add a custom banner, which will get displayed at the top of the page.
This feature can be used to communicate messages to all users, e.g., when external services are experiencing outages.
Administrators can add banners by using the input mask in the admin panel.

![Adding banner](../banner-media/banner-input.jpg)

![Banner](../banner-media/banner.jpg)

!!! tip
    "Dismissible" controls whether the banner can be dismissed by users via the “x” button in the top-right corner.
