# Recommended Storages

DAMAP allows administrators to configure storage recommendations that are presented to users.
This can be done easily by using the input mask in the admin panel.

![Recommended storage input mask](../storages-media/storage-input.jpg)

Observe that it is necessary to choose a language code.
These should match the ones chosen for translations as described in the [Text Customizations](../texts).
To add further storage translations, click on the three-dots icon next to the storage you want to translate and choose "Translations".

![Storage translation added](../storages-media/storage-translation-added.jpg)

After an administrator has added storages, they will appear as recommended options for all users:

![Recommended storage](../storages-media/recommended-storage.jpg)
